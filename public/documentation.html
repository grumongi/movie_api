Movie API Documentation

1. Get All Movies

Description: Returns a list of all movies.

URL: /movies

Method: GET

Example Response:

[{"Title": "Gladiator", "Year": 2000, "Genre": { "Name": "Action" }, "Director": { "Name": "Ridley Scott" } }]

2. Get a Specific Movie by Title

Description: Returns details of a specific movie.

URL: /movies/:title

Method: GET

Example Request: /movies/Gladiator

Example Response:

{"Title": "Gladiator", "Year": 2000, "Genre": { "Name": "Action" }, "Director": { "Name": "Ridley Scott" }}

3. Get Movies by Genre

Description: Returns movies of a specific genre.

URL: /movies/genre/:genreName

Method: GET

Example Request: /movies/genre/Action

Example Response:

{"Name": "Action"}

4. Get Director Details

Description: Returns details about a director.

URL: /movies/directors/:directorName

Method: GET

Example Request: /movies/directors/Ridley Scott

Example Response:

{"Name": "Ridley Scott"}

5. Add a New User

Description: Creates a new user account.

URL: /users

Method: POST

Example Request Body:

{"name": "John Doe"}

6. Add a Movie to a User’s Favorite List

Description: Adds a movie to a user's favorite list.

URL: /users/:id/:movieTitle

Method: POST

Example Request: /users/1/Gladiator

7. Update a User’s Name

Description: Updates a user’s name.

URL: /users/:id

Method: PUT

Example Request Body:

{"name": "Michael Johnson"}

8. Remove a Movie from a User’s Favorite List

Description: Removes a movie from a user's list.

URL: /users/:id/:movieTitle

Method: DELETE

9. Delete a User

Description: Deletes a user account.

URL: /users/:id

Method: DELETE

Final Notes

Status Codes:

  • 200 OK - Request successful
  • 201 Created - Resource created
  • 400 Bad Request - Client error
  • 404 Not Found - Resource not found
