Movie API Documentation

1. Get All Movies

Description: Returns a list of all movies.

URL: /movies

Method: GET

Example Response:

[{"_id": "65f4e2d5a34b9a0012345678", "Title": "Gladiator", "Description": "A general is betrayed and seeks revenge", "Genre": { "Name": "Action", "Description": "Fast-paced movies with physical feats" }, "Director": { "Name": "Ridley Scott", "Bio": "British film director and producer." } }]

2. Get a Specific Movie by Title

Description: Returns details of a specific movie.

URL: /movies/:title

Method: GET

Example Request: /movies/Gladiator

Example Response:

{"_id": "65f4e2d5a34b9a0012345678", "Title": "Gladiator", "Description": "A general is betrayed and seeks revenge", "Genre": { "Name": "Action", "Description": "Fast-paced movies with physical feats" }, "Director": { "Name": "Ridley Scott", "Bio": "British film director and producer." } }

3. Get Movies by Genre

Description: Returns movies of a specific genre.

URL: /movies/genre/:genreName

Method: GET

Example Request: /movies/genre/Action

Example Response:

[{"Title": "Gladiator", "Genre": { "Name": "Action" }}]

4. Get Movies by Director

Description: Returns movies by a specific director.

URL: /movies/directors/:directorName

Method: GET

Example Request: /movies/directors/Ridley Scott

Example Response:

[{"Title": "Gladiator", "Director": { "Name": "Ridley Scott" }}]

5. Add a New User

Description: Creates a new user account.

URL: /users

Method: POST

Example Request Body:

{"Username": "JohnDoe", "Password": "password123", "Email": "john@example.com", "Birthday": "1990-01-01"}

6. Get All Users

Description: Returns a list of all users.

URL: /users

Method: GET

7. Get User by Username

Description: Returns details of a specific user.

URL: /users/:Username

Method: GET

Example Request: /users/JohnDoe

8. Update a User

Description: Updates a user’s information.

URL: /users/:id

Method: PUT

Example Request Body:

{"Email": "newemail@example.com"}

9. Add a Movie to User’s Favorite List

Description: Adds a movie to a user's favorite list.

URL: /users/:id/:movieId

Method: POST

10. Remove a Movie from User’s Favorite List

Description: Removes a movie from a user's favorite list.

URL: /users/:id/:movieId

Method: DELETE

11. Delete a User

Description: Deletes a user account.

URL: /users/:id

Method: DELETE

Final Notes

Status Codes:

  • 200 OK - Request successful
  • 201 Created - Resource created
  • 400 Bad Request - Client error
  • 404 Not Found - Resource not found
